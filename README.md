# PyCharm Projects

A collection of small Python projects and practice scripts.  
Each folder is a separate mini-project and contains its own README.md with a section of what I learned. 

## Projects

- `pink_simple_calculator/`
  - **Description:** A very pink calculator GUI built with `tkinter` that supports basic arithmetic (`+`, `-`, `*`, `/`), decimals, and a clear function.
  - **Main file:** `simplecalculator.py`
 
- `simple_text_editor/`
  - **Description:** A basic `tkinter` text editor that can create, open, edit, and save text files, with options to change font, size, and text color.
  - **Main file:** `texteditor.py`

- `tic_tac_toe/`
  - **Description:** A two-player Tic Tac Toe game built with `tkinter` that lets players take turns on a 3Ã—3 grid, highlights the winning line, and detects ties.
  - **Main file:** `tictactoe.py`

- `snake_game/`
  - **Description:** A classic Snake game built with `tkinter` where the snake moves on a grid, eats food to grow, tracks score, detects collisions, and allows restarting after game over.
  - **Main file:** `snake.game.py`
